Method untuk menentukan metode pengelohan form. pilihan method ada dua yaitu GET dan POST. GET digunakan untuk mengirim data melalui URL, sedangkan POST digunakan untuk mengirim data melalui body permintaan HTTP. Get Data kelihatan di alamat web (URL) biasanya digunakan untuk data yang tidak bersifat rahasia atau untuk pencarian, sedangkan Data tidak kelihatan di URL biasanya digunakan untuk data bersifat rahasia atau privasi.
